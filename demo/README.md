
## Shirow basic demo

### Requirements
 - Python 3
 - NPM

### Install and run

``` bash
$ git clone https://github.com/tolstoyevsky/shirow.git
$ cd shirow
$ npm install
$ npm run demo
$ cd demo
$ pip3 install -r requirements.txt
$ python3 demo.py
```

Link for browser: [http://localhost:8888/](http://localhost:8888/)
